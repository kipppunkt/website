The `npx` command used in [Start in 5 minutes](/get-started/run-kipppunkt/) requires no installation. For regular use, install kipp•punkt permanently.

## npm (recommended)

```bash
npm install -g @kipppunkt/agent
```

This lets you run `kipppunkt-agent` directly from your terminal.

## Direct download

If you prefer not to use Node.js, download the binary for your platform from [GitHub Releases](https://github.com/kipppunkt/build/releases) and place it on your `PATH`.
